Sensor fusion library for Skyduck & Snowduck mobile apps

Skyduck and Snowduck are the mobile tracking apps for skydivers and skiers / snowboarders, respectively.

I've developed a data processing library for these apps, which processess the live data from the mobile device sensors: GPS, barometer, accelerometer, magnetometer and gyroscope.

This library solves two major problems:

  1. Improve the raw GPS tracks quality and resolution using the data from other sensors. I've designed and implemented a custom data fusion algorithm, as conventional algorithms like Kalman filter didn't quite fit our needs.
  2. Detect domain-specific user activities: free-fall, canopy and landing for skydivers; downhill, lift and rest for skiers. Here I've used conventional ML approaches to solve the activity classification problem.

The prototyping and R&D was done using Python + Numpy, Pandas, Matplotlib and other scientific libraries. The production solution is a library written in C++.
